# Drasi Server Tutorials

Each folder in this directory is a self-contained tutorial. Start with **Getting Started** and
work through the others as they become available.

| Tutorial | Description |
| --- | --- |
| [getting-started](getting-started) | Create your first Drasi Server pipeline: a mock source, a continuous query, and a log reaction. |

## Adding a new tutorial

Copy the [`TEMPLATE`](TEMPLATE) folder, rename it, and fill in the content. See
[CONTRIBUTING.md](../CONTRIBUTING.md) for the conventions every tutorial should follow.
