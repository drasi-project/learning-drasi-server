Drasi Server Tutorials

Drasi Server is a lightweight, single-process Drasi deployment that provides the full power of Drasi’s data change processing functionality. Perfect for local development, Docker environments, and small-scale deployments.

This repository and documentation site host the hands-on tutorials for Drasi Server. The instructions here walk you through connecting Drasi Server to live data sources and building Continuous Queries and Reactions that detect and respond to change in real time.

Start with the Getting Started tutorial to stand up a running Drasi Server reacting to a live PostgreSQL database.
